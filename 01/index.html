Insertar JS dentro de HTML

Se usa la etiqueta script y se puede escribir código JS dentro de ella o dejarla vacía y llamar a un archivo .js usando el atributo src.

Sintaxis de JS.

  • Case Sensitive (sensible a las mayúsculas y minúsculas)
  • Es de tipado dinámico
  • Los comentarios en JS de pueden escribir de dos maneras:
    • /* */ (multilínea)
    • // (una sola línea - recomendada)
  • Todas las sentencias deben terminar con punto y coma
  • Por regla general, después de llaves no debe ir punto y coma, salvo dos excepciones:
    • Cuando se crea un obejto literal
    • Cuando se crea una funcion con expresiones

Instalar SublimeLinter JsHint (ST3)

  • Buscar e instalar SublimeLinter desde el package control
  • Instalar SublimeLinter-jshint
  • Instalar NodeJS desde su página web (http://nodejs.org)
  • Instalar jshint desde npm con el siguiente comando en terminal: npm install -g jshint

Variables

Son contenedores de información.

Se crean con la palabra reservada "var" seguida del nombre la variable y con el signo = se le asigna un valor.

Se podría también crear una variable y no asignarle ningún valor. En este caso al variable devuelve undefined.

Las variables tienes un scope (ámbito) en el cual existen. Fuera de ese ámbito no existen.

Toda variable que se declara dentro de una función, tiene a esa función com su scope.

Toda variable declarada desde fuera de cualquier función se llama variable global (no es buena práctica usarlas). Además crea una variable sin "var" la vuelve automáticamente global, lo cual no es bueno.

Tipos de Datos

  • Primitivos (un solo valor)
    • Primarios
      • Number
      • String
        son cero o más caracteres encerrados entre comillas (dobles o simples). Es caracter de escape es el backslash (\).
      • Boolean
        Sólo tienen dos valores posibles: true (verdadero) y false (falso). Existen valores que se comportan como false (falsy values) y otros que se comportan como true (truthy values).
        • Falsy Values:
          • NaN
          • 0
          • ""
          • Null
          • undefined
    • Especiales
      • undefined
      • Null
  • Compuestos (más de un valor)
    • Arrays
    • Objetos

Operadores

Signos que se utilizan para realizar operaciones.

  • Operadores de asignación
    • =
    • +
    • -
  • Operadores de comparación
    • == Igualdad de valor
    • === Igualdad de valor y tipo de dato (reomendada)
    • != No igualdad de valor
    • !== No igualdad de valor y tipo
    • < Menor que
    • > Mayor que
    • <= Menor o igual que
    • >= Mayor o igual que
  • Operadores Aritméticos
    • + Suma
    • - Resta
    • * Multiplicación
    • / División
    • % Módulo
    • ++ Incremento
    • -- Decremento
