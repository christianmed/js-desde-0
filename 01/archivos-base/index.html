JavaScript Desde Cero - Clase 1

Insertar JavaScript dentro de HTML

Se usa la etiqueta <script></script> y se puede escribir código javascript dentro de ella, o dejarla vacia y llamar a un archivo .js externo usando el atributo src.

Sintaxis de JavaScript

  • Case sensitive (es sensible a las mayúsculas y minúsculas)
  • Es de tipado dinámico.
  • Los comentarios en JS se pueden escribir de dos maneras:
    • /* */ (multilinea)
    • // (para cada linea, recomendada)
  • Todas las sentencias deben terminar con punto y coma.
  • Por regla general, después de llaves no debe ir punto y coma, salvo dos excepciones:
    • Cuando se crea un objeto literal
    • Cuando se crea una funcion con expresiones

Variables

Son contenedores de información.

Se crean con la palabra reservada "var" seguida del nombre de la variable (el que queramos) y con el signo igual se le asigna un valor.

Se podria también crear una variable y no asignarle ningun valor. En este caso la variable devuelve "undefined"

Las variables tienen un scope (ambito) que es el "ambito" en el cual existen. Fuera de ese ambito, no existen.

Toda variable que se declara dentro de una funcion tiene a esa funcion como su scope

Toda variable declarada desde fuera de cualquier funcion se llama variable global (no es una buena practica usarlas). Ademas crear una variable sin "var" la vuelve automáticamente global.

Tipos de datos

  • Primitivos (un solo valor)
    • Primarios
      • Number
        JavaScript no hace diferencia entre enteros, decimales, negativos, etc.
      • String
        Son cero o más caracteres encerrados entre comillas. Pueden ser comillas simples o dobles, lo importante es que se correspondan. El caracter de escape es el backslash (\)
      • Boolean
        Solo tienen dos valores posibles: true (verdadero) y false (falso). Existen valores que se comportan como false (falsy values) y otros que se comportan como true (truthy values).
        Falsy values:
        • NaN
        • 0
        • ""
        • Null
        • Undefined
    • Especiales
      • Undefined
      • Null
  • Compuestos (más de un valor)
    • Arrays
    • Objetos

Operadores

Signos que se utilizan para realizar operaciones

  • Operadores de asignacion
    • =
    • +=
    • -=
  • Comparacion
    • == (igualdad de valor)
    • === (igualdad de valor y tipo, recomendada)
    • != (no igualdad de valor)
    • !== (no igualdad de valor y tipo)
    • < Menor que
    • >
    • <=
    • >=
  • Operadores aritméticos
    • + (en el caso de numeros, hace una suma, para strings los concantena.
    • -
    • *
    • /
    • % Modulo: devuelve el resto de una division
    • ++ Aumenta en uno (incremento). Si es antes es un preincremento (incrementa antes de retornar el valor). Y si es despues es un postincremento (incrementa después de retornar el valor)
    • -- Disminuye en 1 (decremento)
