Cadenas de texto

Para saber la longitud de una cadena de texto o la cantidad de caracteres se puede usar la propiedad .lenght.

Para saber si una cadena de texto tiene un caracter específico y de tenerlo saber su posición, usamos el método .typeOf().

Para pasar el texto de una cadena a mayúsculas, usamos el método .toUpperCase().

Para pasar el texto de una cadena a minúsculas, usamos el método .toLowerCase().

Para reemplazar elementos dentro de una cadena de texto, usamos el método .replace(). La cadena de texto original no se modifica.

Para extraer elementos dentro de una cadena de texto, usamos el método .substring(). Dicho método puede recibir dos parámetros, la posición de inicio y la posición donde de detiene. Se le pude pasar solo el parámetro de inicio. Este método no acepta números negativos. La cadena de texto original no se modifica.

El método .slice() se comporta como el método substring si se le pasan parámetros positivos. Si se le pasan parámetros negativos (los acepta) inicia desde el final. La cadena de texto original no se modifica.

El método .trim() elimina los espacios al inicio y al final de una cadena de texto.

Con el método .split() convertimos una cadena de texto en array. Se le pasa como parámetro a partir de qué va a separar la cadena.

Arrays

Son un grupo de valores encerrados entre corchetes ([]) y separados por comas (,). Ejemplo: var array = [1,2,3,"Hola","Mundo",true]. Cada valor es un elemento del array y podemos determinar su longitud de la misma forma que con las cadenas de texto.

Para recuperar un elemento dentro de un array usamos lo siguiente: array[1], array[3].... Eso nos devuelve el segundo y tercer elemento ya que en JS el primer índice es el 0, el segundo es el 1, etc.

Para agregar un nuevo elemento o sobrescribir elementos existentes usamos lo siguiente: array[índice] = valor.

Con el método .join() convertimos un array en string.
